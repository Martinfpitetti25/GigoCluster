Pensamiento de IA desde el inicio: ejercitá programación y depuración (debugging) con desafíos guiados; aprendé a usar la IA para trabajar mejor, optimizando pasos, corrigiendo errores y creando rutinas más inteligentes.

Preguntas Frecuentes

Encuentra respuestas a las consultas más comunes sobre nuestros kits y servicios

¿Qué edades están recomendadas para cada kit?

Cada kit está diseñado para diferentes rangos de edad: Kit 1205 (6-10 años), Kit 1269 (10-14 años), Kit 1300 (12-16 años), Kit 1409 (14+ años), y Kit 1204RR (12+ años). Todos están pensados para facilitar el aprendizaje progresivo.

¿Necesito conocimientos previos de programación?

No es necesario. Gigo se encargó de que los kits estén diseñados para iniciar desde cero. Incluyen guías paso a paso, programación visual por bloques y materiales educativos que facilitan el aprendizaje de forma intuitiva.

¿Realizan envíos a todo el país?

Sí, realizamos envíos a todo el territorio nacional. Los tiempos de entrega varían según la ubicación. Contáctanos para obtener información específica sobre costos y plazos de entrega para tu zona.

¿Qué garantía tienen los productos?

Todos nuestros kits cuentan con garantía de 6 meses contra defectos de fabricación. Además, ofrecemos soporte técnico continuo y asesoramiento para sacar el máximo provecho de tu kit educativo.

¿Incluyen material didáctico y guías?

Absolutamente. Cada kit incluye guías de actividades, desafíos progresivos y acceso a secuencias pedagógicas. También ofrecemos soporte pedagógico.

¿Ofrecen capacitación para docentes?

Sí, ofrecemos programas de capacitación y talleres especializados para docentes. Incluyen formación en uso de kits, metodologías pedagógicas y estrategias para integrar la robótica en el aula.

¿Se requiere una computadora para usar los kits?

Depende del kit. El Kit 1205 no requiere PC. Los kits 1269, 1300, 1409 y 1204RR sí necesitan computadora para programación. Algunos también soportan tablets o dispositivos móviles con Bluetooth.

¿Qué formas de pago aceptan?

Aceptamos transferencia bancaria, depósito, tarjetas de crédito y débito. Para instituciones educativas ofrecemos planes de pago especiales y presupuestos personalizados. Contáctanos para más detalles.
